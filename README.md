# Runnning Coach MVP

THIS IS A WORK IN PROGRESS
Right now, only the backend upload endpoint exists.

This is an experimental personal project to learn about AI agents and build an AI-powered running coach.
The goal is to have an app that allows you to upload FIT files from a GPS watch, extract metricts, and get AI-generated running advice based on your training data.

This project is mainly for learning:
  FastAPI
  Data ingestion
  Fitness data analysis
  React Native (planned)
  AI agents & LangChain (future)
The end goal is a full AI training assistant that analyzes your runs and gives coaching feedback conversationally.
